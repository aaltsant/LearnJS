# LearnJS! - Fullstack course project

This project was a course project for my fullstack course! It is an application where you can train your JavaScript knowledge and compete against other users and get your name to the leaderboard.
I had an idea about making this type of project way before this course but never knew how and after this course I had all the needed information for making this project!

App features:
  - Training Mode:
    - Here you can train your knowledge and get ready to try the test!
  - Test Mode:
    - Here is 15 questions test with streak system and automatic point saving system:
  - Admin page:
    - Page where you can Create, Read, Update and Delete easily from web.
